# react
    * 컴포넌트기반의 자바스크립트 프레임워크
    * 컴포넌트는 function 기반 컴포넌트와 class기반 컴포넌트로 나눔
    * 주로 function 기반의 컴포넌트를 사용함
    * index.js : 엔트리 컴포넌트, 모든 컴포넌트의 시작 컴포넌트
    * 컴포넌트가 컴포넌트(들)을 import후 조합하여 사용
    * import해야하는 컴포넌트들은 export를 해야 사용 가능
    * 컴포넌트내에서는 JSX문법을 사용
    * JSX문법은 빌드 과정에서 Babel를 통해 JS로 변환된다.

# 빌드와 실행
    * 빌드 : 컴포넌트들과 관련파일들(CSS, 이미지 ...)을 묶어서 실행가능한 상태로 변경하는 것
        - npm run build
    * 실행 : 빌드된 것을 실행
        - npm serve -s build
